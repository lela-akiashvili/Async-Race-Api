## Getting Started

To run Epam-Async-Race-Api, follow these steps:

1. Clone the Repository:

```
git clone https://github.com/lela-akiashvili/eslint.git
cd eslint
```
2. Install dependecies:

```
npm install
```
3. Run ng serve:

```
ng serve -o
```
